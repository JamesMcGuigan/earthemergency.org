This is a sample interactive html form which uses Matt Wright's FormMail script v1.6, a generic WWW form to e-mail gateway. The CGI script allows webmasters to create interactive forms to process feedback from their web site visitors either to place orders or ask for additional information. In this example, the script will parse the results of this sample form and send them to the specified user, typically the webmaster or the owner of a particular business.

For more information on Matt Wright's scripts, please refer to the documentation which can be found at: http://www.worldwidemart.com/scripts

To see a sample html form which demonstrates the additional features in the BFormMail CGI script, including courtesy replies, e-mail to fax and append to database, please look at the BFormMail sample form, or reference its documentation at http://www.infosheet.com/iScripts.html

Be sure to enter correct e-mail addresses if you wish to receive the form results. Typically, the recipient field would be a hidden field within the html file.
